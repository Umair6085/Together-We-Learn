CSS Types

This page demonstrates three types of CSS:

  • Inline CSS: Applied directly to elements using the style attribute.
  • Internal CSS: Defined within the <style> tag in the <head> section.
  • External CSS: Linked using an external stylesheet (not used here, but you can try).
